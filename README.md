# Sistema-de-Gesti-n-de-Reservas-y-Pagos
Plataforma web completa para la gestión, consulta y formalización de reservas, integrando un flujo de pago simulado.  

Desarrollo del backend en PHP para la lógica de negocio, gestionando el alta de reservas, validaciones y consultas estructuradas a la base de datos (vconsultas.php, vreservas.php).  

Diseño e implementación de la base de datos relacional en MySQL para el almacenamiento de la información de los usuarios y transacciones (reservas.sql).  

Integración de la pasarela de pago Redsys (entorno de pruebas) para simular transacciones seguras y completar el ciclo de reserva (fun-pagar.php).  

Maquetación de la interfaz de usuario utilizando Bootstrap para asegurar un diseño responsive y una navegación intuitiva.
